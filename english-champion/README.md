# 🏆 English Champion
### คู่มือเรียนภาษาอังกฤษแบบโต้ตอบ
Interactive English Study Guide for Thai Students

## Deploy to Vercel (Step by Step)

### Option A — Drag & Drop (Easiest, No Account Needed for Dev)
1. Go to vercel.com and sign up free (use Google or GitHub)
2. Click "Add New Project"
3. Click "Upload" and drag this entire folder
4. Vercel auto-detects React — just click Deploy
5. Your live URL appears in seconds! 🎉

### Option B — GitHub (Recommended for updates)
1. Create free account at github.com
2. Create new repository called "english-champion"
3. Upload all files from this folder
4. Go to vercel.com → "Add New Project" → Import from GitHub
5. Select your repo → Deploy
6. Every time you update files on GitHub, Vercel auto-redeploys!
